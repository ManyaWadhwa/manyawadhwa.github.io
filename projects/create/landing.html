CREATE: Creative Associations Test for LLMs

CREATE is a benchmark for evaluating language models on creative association tasks. The benchmark focuses on testing how well LLMs can generate creative connections and pathways between concepts. It includes multiple prompting strategies, including original and creative approaches, to assess the model's ability to think creatively and make novel associations. The dataset contains numerous examples with queries, source paths, and predictions, providing a comprehensive evaluation framework for creative capabilities in language models.
